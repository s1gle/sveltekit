Особенности реализации:

Сетка создается с использованием CSS Grid Layout

52 колонки (недели) × 7 строк (дни недели)

Цвета соответствуют оригинальной схеме GitHub

При наведении показывается всплывающая подсказка с датой и количеством

Анимация при наведении (увеличение элемента)

Данные передаются в компонент через props

Автоматическая инициализация дат за последние 52 недели

Адаптивная верстка

Для использования:

Передайте массив объектов с полями date (строка в формате YYYY-MM-DD) и count

Стили можно настроить через CSS-переменные или модифицировав стили компонента

Можно добавить дополнительные параметры (размер ячейки, цвета) через props

Основные изменения:

Структура данных:

Год разделен на 4 квартала по 3 месяца

Для каждого месяца рассчитываются недели

Учитываются дни из предыдущего/следующего месяца в первой/последней неделе

Визуальные изменения:

Кварталы отображаются вертикально

Месяцы в квартале расположены горизонтально

Заголовки месяцев над каждым месяцем

Дни вне текущего месяца подсвечиваются прозрачным цветом

Логика отображения:

Автоматическое определение границ месяцев

Корректное отображение "захваченных" дней соседних месяцев

Адаптивная сетка для каждого месяца

Стили:

Уменьшен размер ячеек для компактного отображения

Добавлены промежутки между кварталами

Улучшена анимация при наведении

Особенности работы:

Автоматически определяет текущий год

Подсвечивает только дни текущего месяца

Сохраняет оригинальную цветовую схему GitHub

Отображает кварталы в виде вертикальных блоков с горизонтальным расположением месяцев
